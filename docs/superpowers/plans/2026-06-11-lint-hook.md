# Pre-commit Lint Hook Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add ESLint + husky + lint-staged so staged TypeScript/TSX files are linted (and auto-fixed) on every commit.

**Architecture:** ESLint flat config at repo root covers all `apps/**/*.{ts,tsx}` with TypeScript rules, plus React hooks rules for the frontend. husky registers a pre-commit hook that runs `pnpm lint-staged`. lint-staged passes only staged files to ESLint, keeping commits fast.

**Tech Stack:** eslint ^9, @eslint/js, typescript-eslint, eslint-plugin-react-hooks, husky, lint-staged

---

## File Map

| File | Action | Responsibility |
|------|--------|---------------|
| `eslint.config.js` | Create | ESLint flat config — TS + React hooks rules |
| `package.json` | Modify | Add `prepare` script, `lint-staged` config, devDependencies |
| `.husky/pre-commit` | Create (by husky init) | Runs `pnpm lint-staged` |

### Task 2: ESLint flat config

**Files:**
- Create: `eslint.config.js`

- [ ] **Step 1: Create `/Users/gobobin/projects/project-with-harness/eslint.config.js`**

```js
import js from '@eslint/js'
import tseslint from 'typescript-eslint'
import reactHooks from 'eslint-plugin-react-hooks'

export default tseslint.config(
  js.configs.recommended,
  ...tseslint.configs.recommended,
  {
    files: ['apps/**/*.{ts,tsx}'],
    languageOptions: {
      parserOptions: {
        projectService: true,
        tsconfigRootDir: import.meta.dirname,
      },
    },
  },
  {
    files: ['apps/frontend/**/*.tsx'],
    plugins: {
      'react-hooks': reactHooks,
    },
    rules: reactHooks.configs.recommended.rules,
  },
  {
    ignores: ['**/dist/**', '**/node_modules/**'],
  },
)
```

- [ ] **Step 2: Verify ESLint runs without errors on the existing codebase**

```bash

```

Expected: no output (no errors). If there are errors, fix them before continuing.
